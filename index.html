Your Best Game

By Oscar Duran Marti / GitHub

Que es YourBestGame?

1- Comunitat per comprartir els teus jocs

2- Pagina per passar un rato divertit

3- Donar a coneixer la teva idea

4- Ajuda als creadors per a millorar


Desenvolupada amb Laravel, Vue.js, Bootstratp i més tecnologies

Desenvolupacio

  • Llenguatges usats: PHP - Javascript - HTML5 - CSS3 - Java
  • Tecnologies usades:
    • API: Laravel - Bootstrap - Eloquent - MySQL
    • FrontEnd: Bootstrap - Vue.js - Ajax - jQuery
  • llibreries i Dependències
    • API: AdminLTE - ApiGuard
    • FrontEnd: Vue-router - Vue-resource

Funcionalitats de YourBestGame (FrontEnd)

  • Registre d'usuaris que es realitzarà sobre la API per més seguretat.
  • Els usuaris sense necessita de registrar se podran veure els jocs penjats per altres usuaris.
  • Podran compartir el seu joc fàcilment.
  • Podran consultar els seus jocs al menú.

Funcionalitats de YourBestGame (BackEnd / API)

  • Es on tenim diseñada la API.
  • Sera el lloc on estaran registrades les rutes i els controladors.
  • Amb l'ApiGuard controlarem les peticions.
  • També controlem les dades entrants(Request).
  • Dins l'API tenim un 'Transofmer' que ens depura les dades.
